### chat-app-socketio
chat app using react and socket io

## Install

#### For frontend with React
> 1. Clone repository
> 2. use in shell "npm install"
> 3. Use in shell "npm run start"

#### For backend in node with socket io
> 1. cd /backend folder
> 2. use in shell "npm install"
> 3. use in shell "node app.js"
